Simple Paint App

Using (chrome compatible browser)

  1. Module 1 :

    • (Left click down + Mousemove) :- draw rectangle.
    • (Clear button) :- clear/reset canvas.

  2. Module 2 :

    • (Left click down + Mousemove) :- drag rectangle for existing rectangle.
    • (double click) :- delete rectangle.

  3. Extra :

    • (Left click down + Mousemove) :- resize rectangle for existing rectangle.
    • Note:- Resize work from border to 5px inside rectangle.

  4. Automatically :

    • Change different color.
